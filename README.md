Welcome to my DSA preparation repository! This repository contains clean, well-documented Python implementations of core data structures and algorithms, designed for learning, practicing, and reference.

## 📂 Repository Structure

* 📁 **[dsa-prep](file:///Users/gauraverma/src/dsa-prep)**
  * 📄 **[dynamic_array.py](file:///Users/gauraverma/src/dsa-prep/dynamic_array.py)**: A custom dynamic array (resizable array) implementation from scratch in Python, demonstrating:
    * Amortized $O(1)$ pushback.
    * Automatic capacity doubling (resizing).
    * $O(N)$ element insertion at any index with elements shifting.
    * $O(1)$ popback and indexing.
    * Custom string representation (`__repr__`) for easy debugging.

## 🛠️ How to Run the Code

Each data structure file includes a self-contained test/demonstration block. You can execute any file directly using Python 3:

